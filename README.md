# Helvia

A professional B2B SaaS landing page for an AI receptionist platform. Built as a portfolio project to showcase frontend development skills.

## Overview

Helvia is a fictional B2B SaaS platform designed for modern businesses. It handles customer conversations 24/7, routes messages intelligently, and helps teams respond faster without extra workload.

## Tech Stack

- **React** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Vite** - Build tool and dev server

## Features

- Fully responsive design
- Smooth animations and transitions
- Professional B2B SaaS UI
- Sticky navigation with scroll effects
- Interactive dashboard mockup
- Contact form (UI only, no backend)
- Modern, clean design following B2B SaaS best practices

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
